# Gravity Blast

An arcade puzzle game where gravity rotates after every move. Place polyomino pieces on an 8×8 board, group same-colored blocks into clusters of 6+, and blast them for points. Chain clears build a combo multiplier that carries across turns.

## Play

👉 **[Play in your browser](https://ekyritsi.github.io/gravity-blast/)**

## Local development

No build step. Just open `index.html` in a browser.

## Files

- `index.html` — markup, HUD, board, tray
- `style.css` — theme, animations, mobile layout
- `script.js` — game logic (config, gravity, clustering, animation)
